# Number Sequences (Generator)
#### Video Demo:  https://youtu.be/Nq3DWe8-stQ

#### Description:

This project is a simple and modern web application designed to generate various types of mathematical sequences. The web app allows users to easily create and visualize three fundamental mathematical sequences:

* **Arithmetic Progression (AP)**
* **Geometric Progression (GP)**
* **Fibonacci Sequence**

The application is built using fundamental web technologies like **HTML**, **CSS**, **JavaScript**, and **Bootstrap**. Additionally, it uses **MathJax** to render mathematical equations neatly, making the interface user-friendly and visually appealing. The app is hosted on **GitHub Pages**, ensuring that it's easy to access and use on any device with a web browser. This project was developed as part of my final submission for the **CS50** course, which is known for its rigorous and comprehensive approach to computer science.

With this web app, users can instantly generate sequences by simply providing the necessary inputs like the first term, common difference (for AP), common ratio (for GP), or the number of terms (for Fibonacci). This app makes learning about these sequences much more interactive and hands-on.

### How to Use the App:

1. Choose the type of sequence you want to generate: AP, GP, or Fibonacci.
2. Enter the required input values (like the first term, common difference, common ratio, or number of terms).
3. Click on the "Generate Sequence" button.
4. The sequence will be displayed instantly on the page.
5. You can download the result or copy it to your clipboard for further use.
