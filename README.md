# Number Sequences (Generator)
#### Video Demo:  https://youtu.be/Nq3DWe8-stQ
#### Description:
A simple and modern web application to generate mathematical sequences such as:

- Arithmetic Progression (AP)
- Geometric Progression (GP)
- Fibonacci Sequence

Built using **HTML, CSS, JavaScript, and Bootstrap**, and hosted using **GitHub Pages**. <br>
I have done this as cs50 final project.
